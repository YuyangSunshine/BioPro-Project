Abstract

Automation in biological experimentation is currently bottlenecked by the "safety-critical reasoning gap" in Large Language Models (LLMs). Hallucinations in wet-labs can lead to hazardous failures.

To solve this, we present a unified ecosystem comprising two key contributions: (1) BioProBench, a dataset of 550k task instances derived from 27,000 human protocols to benchmark procedural reasoning; and (2) BioProAgent, a neuro-symbolic agent that uses a finite state machine (FSM) to ground LLM planning in physical reality.

BioProAgent addresses the probabilistic nature of LLMs by introducing State-Augmented Planning. Unlike standard agents that generate open-ended text, our system simulates the state of lab equipment (e.g., Temperature, Volume) to verify every step.

Figure 1: The BioProAgent Architecture. The LLM (Neuro) proposes high-level plans, while the FSM (Symbolic) acts as a verifier to enforce hardware constraints (e.g., ensuring a PCR machine is on before cycling).

Key Results

By using Semantic Symbol Grounding, we reduce token consumption by ~6x compared to full-text schemas. The agent achieves a significantly higher success rate on the BioProBench test set compared to GPT-4 baselines.
