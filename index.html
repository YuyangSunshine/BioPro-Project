研究摘要

大型语言模型（LLMs）在处理对准确性要求极高的生物实验草案（Protocols）时缺乏可靠性，这严重阻碍了科学实验的自动化进程。 为此，我们首先引入了包含 55 万任务实例的 BioProBench 基准测试，以揭示现有模型的推理鸿沟；进而提出了神经符号智能体框架 BioProAgent。 通过将概率性的规划过程锚定在确定性的有限状态机（FSM）中，我们的智能体能够确保硬件操作的物理合规性，并在性能上显著超越了以 GPT-4 为代表的基准模型。

🧬 数据集: BioProBench

我们提出了 BioProBench，这是首个专注于生物实验草案过程推理的大规模资源库。它包含一个由近 27,000 份专业实验草案组成的语料库 (BioProCorpus)，以及由此衍生出的超过 550,000 个结构化实例，覆盖了生物学的多个子领域。

BioProBench 概览： (a) 包含 27,000 份专业草案的基础语料库； (b) 衍生出的 550,000+ 结构化数据集，已划分为训练集（用于模型微调）和保留测试集； (c) 一个严谨的基准测试，引入了基于关键词内容和基于嵌入结构的新型特定领域评估指标，旨在准确量化程序性知识的掌握程度。

Extend-BioProBench 评测

我们在 BioProBench 基础上构建了扩展评测集（extend-bioprobench），包含四个子集： Subset A（Protocol Drafting）评估科学有效性； Subset B（Code Generation）评估硬件 schema 对齐与参数准确性； Subset C（Long-Horizon）评估全局编排能力（9 个复杂 protocol，共 547 个原子步骤，单任务最多 238 步）； Subset D（Error Correction）评估注入错误下的鲁棒性。 为保证可复现性，我们提供完整代码、覆盖 22 类合成生物学核心仪器的数字化硬件注册表（Ω）以及数据集。 虽然在仿真环境中评估，但所有生成指令都严格遵循标准自动化设备 API 规范，以缩小 sim-to-real gap。

🤖 核心方法: BioProAgent

BioProAgent：用于受限科学规划的神经符号接地框架

  • 状态增强的自适应规划（受 FSM 约束的无脚本规划器）： 摒弃了僵化的线性工作流，采用神经符号框架，通过确定性的有限状态机（FSM）来约束概率性的模型规划。智能体能够根据当前状态灵活选择检索、草案生成或代码编写，有效解决了 LLM 在处理湿实验严苛物理约束时的短板。
  • 科学审查（Scientific Review）： 内置了严格的科学反思机制（验证器），自动检查对照组遗漏、逻辑缺陷、参数合理性和机器代码有效性。强制执行严谨的“设计-验证-纠错”（DVR）工作流，确保实验草案的科学严密性。
  • 自动化硬件对齐： 读取实验室设备和耗材库存（CSV），通过语义符号接地（Semantic Symbol Grounding）将自然语言步骤映射为具体的机器操作，同时将 Token 消耗降低约 6 倍。
  • 混合记忆系统：
    • 短期记忆：结合情景记忆（Episodic）和工作记忆（Working），维持长序列实验草案的一致性。
    • 长期记忆：集成 Mem0 框架，用于回忆过去的实验经验。
  • 人类在环（Human-in-the-Loop）： 在关键决策点主动请求用户确认，为高风险的湿实验操作提供最终安全保障。

BioProAgent 架构概览： (a) 认知记忆模块利用符号接地 Φ 高效管理上下文； (b) 神经规划器 π₀ 被锚定在“设计-验证-纠错”的 FSM 状态机 Δ(σ) 中； (c) 分层验证机制 (Kₛ, Kₚ) 作为安全联锁，通过确定性触发纠错来强制保证物理环境的合规性。

📈 BioProAgent 核心性能表现

BioProAgent 打破了传统智能体在“科学推理”与“物理安全”之间无法兼顾的困局。与最先进的基准模型相比，它在硬件合规性、长序列稳定性以及成本效率方面表现出显著优势：

  • 极高的物理合规性： 达到了 95.6% 的物理操作合规率，成为抵御“模型幻觉”的关键安全联锁机制（相比之下，ReAct 智能体因安全崩溃仅获得 21.0% 的合规率）。
  • 自主纠错与自愈： 当面对注入的错误时，所有标准基准智能体的纠错率均为 0%；而 BioProAgent 通过 FSM 动态覆盖不安全的轨迹，将物理安全恢复率提升至 88.7% 。
  • 极致的成本效率： 通过语义符号接地机制解耦高维数据载荷，在维持 60 步长视野任务 100% 成功率的同时，将 Token 消耗比 AutoGPT 降低了 约 82% 。

图示：科学推理 vs. 自动化可执行性。 原生 LLM 往往位于 理论区（Theoretical Zone），即逻辑推理较强但代码执行能力较弱；神经智能体（如 ReAct）通常代码生成更好，但科学推理能力不足。BioProAgent 在两条维度上同时领先，实现了更高水平的 可信自主（Trustworthy Autonomy）。

🔍 案例展示：FSM 驱动的自我纠错

标准的 LLM 智能体通常在开环状态下运行：一旦生成危险的幻觉参数（例如离心机转速超限），就会直接执行并导致物理损坏。而 BioProAgent 能够主动拦截并修复这些致命幻觉。

图例： 在物理违规案例 (a) 中，符号规则引擎拦截了不安全的超限转速 (25,000g)，强制状态机流转至 RECTIFY_CODE 状态，并重新生成了安全限度内的代码 (15,000g) 。在符号接地错误案例 (b) 中，系统检测到未定义的资源 ID ("new_plate")，并引导智能体将其重新映射至已验证的物理槽位 ("plate_1") 。
