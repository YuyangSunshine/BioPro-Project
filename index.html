Abstract

The automation of scientific experimentation is critically hindered by the inability of Large Language Models (LLMs) to reliably comprehend the specialized, accuracy-critical nature of biological protocols. In wet-lab environments, hallucinations are not merely incorrectâ€”they cause equipment damage.

To address this, we introduce a unified ecosystem:
1. BioProBench (ICLR 2026): A massive-scale benchmark grounded in 27,000 human-written protocols. It exposes the "safety-critical reasoning gap" in current LLMs, showing that even GPT-4 struggles with parameter verification and hardware constraints.
2. BioProAgent (ACL 2026): A neuro-symbolic framework that anchors probabilistic LLM planning in a deterministic Finite State Machine (FSM). By simulating device states (e.g., PCR temperature, pipette volume) before execution, it ensures hardware compliance and zero-hallucination planning.

Current LLM agents often generate protocols that look correct but are physically impossible to execute. BioProAgent solves this by introducing State-Augmented Planning via a Design-Verify-Rectify workflow.

Figure 1: The BioProAgent Architecture. The LLM (Neuro) proposes plans, while the FSM (Symbolic) verifies state transitions to prevent unsafe operations.

Key Innovations

  • Design-Verify-Rectify: Automatically corrects errors (e.g., wrong temperatures) by checking against device specs.
  • Semantic Symbol Grounding: Abstracts complex device schemas into symbols, reducing token consumption by ~6x.

Performance

Significantly outperforms GPT-4 baseline in success rate.
